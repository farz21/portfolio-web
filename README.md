# Portfolio Web - Fabrizio "Farz" Coronel

Este es mi portfolio personal como desarrollador, donde muestro mis proyectos, habilidades y evolución en el mundo de la programación.

##  Sobre el proyecto

El objetivo de este portfolio es centralizar mi trabajo y presentar de forma clara quién soy como desarrollador, qué tecnologías manejo y qué tipo de soluciones puedo construir.

Incluye:
- Proyectos personales y académicos
- Tecnologías que utilizo
- Información de contacto
- Diseño responsive y moderno

## Tecnologías utilizadas

- React
- JavaScript (ES6+)
- HTML5
- CSS3 / [Bootstrap]
- Git & GitHub

##  Características

- Diseño responsive (adaptado a mobile y desktop)
- Navegación simple e intuitiva
- Sección de proyectos con detalles
- Código modular y reutilizable
- Buenas prácticas de desarrollo frontend
